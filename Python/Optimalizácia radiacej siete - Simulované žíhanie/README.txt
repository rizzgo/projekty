V priečinku src je zdrojový kód spustiteľný skriptom stats.py spolu s vykreslením grafov a uložením poľa štatistík (objektov Stats) pre všetky spustené behy. Priečinok "notebooky_pre_experimenty" obsahuje python notebooky používané pre experimentovanie s parametrami a vykresľovanie grafov použitých v dokumentácii. Ich obsahom sú mierne upravené časti kódov zo súborov stats.py a annealing.py. Dokumentácia podrobne popisuje priebeh experimentov.
